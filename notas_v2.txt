*** Verificar a versão atual da sdk do dotnet core.
marcoscrocci@iMac-de-Marcos dotnetcore % dotnet --version
5.0.101
-----------------------------------------------------------------------------------------------
*** Criar o arquivo global.json por linha de comando:
> dotnet new globaljson --sdk-version 5.0.404 (se o arquivo já existir, acrescente --force no final do comando)
-----------------------------------------------------------------------------------------------
*** Criar o projeto WebAPI e entrar na pasta do projeto:
> dotnet new webapi -n ProEventos.API
> cd ProEventos.API/
-----------------------------------------------------------------------------------------------
*** Executar o projeto WebAPI:
> dotnet watch run
-----------------------------------------------------------------------------------------------
*** Gerar o certificado digital do HTTPS:
> dotnet dev-certs https --trust
-----------------------------------------------------------------------------------------------
*** Listar as ferramentas do .NET Core:
> dotnet tool list --global
-----------------------------------------------------------------------------------------------
*** Instalar o Entity Framework CLI:
dotnet tool install --global dotnet-ef
Site de referência: https://docs.microsoft.com/pt-br/ef/core/cli/dotnet
-----------------------------------------------------------------------------------------------
*** Instalar o Entity Framework no Projeto:
Devemos instalar a Extensão NuGet Gallery
Pressionar Command+Shift+P e procurar NuGet: Open NuGet Gallery
Filtrar: EntityFrameworkCore
Selecionar a versão compatível com o .NET Core, o projeto e instalar
Filtrar: EntityFrameworkCore.Design
Selecionar a versão compatível com o .NET Core, o projeto e instalar
Filtrar: EntityFrameworkCore.Tools
Selecionar a versão compatível com o .NET Core, o projeto e instalar
Filtrar: EntityFrameworkCore.SQLite
Selecionar a versão compatível com o .NET Core, o projeto e instalar
